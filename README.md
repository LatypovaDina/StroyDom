# СтройДом - Информационная система для строительства частных домов

## О проекте
Веб-сайт для ООО «СтройДом» - компании, специализирующейся на строительстве частных домов.

## Технологии
- Python 3.x
- Django 4.2.7
- MySQL (через XAMPP)
- HTML5, CSS3, JavaScript

## Функциональность
- 7 основных страниц (главная, о компании, услуги, проекты, новости, контакты, калькулятор)
- Два стиля: стандартный и для слабовидящих (ГОСТ Р 52872-2012)
- Система пользователей с ролями (администратор, менеджер, клиент)
- Поиск по сайту
- Форма обратной связи
- Лента новостей
- Карта сайта
- Обработка ошибок 404
